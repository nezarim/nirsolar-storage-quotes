# ניר סולאר - מערכת הצעות מחיר לאגירת אנרגיה

![Nir Solar](logo.png)

מערכת להפקת ושליחת הצעות מחיר אינטראקטיביות למערכות אגירת אנרגיה.

## תכונות

- **לוח בקרה לנציגי מכירות** - ניהול הצעות, מעקב צפיות
- **דף הצעה אינטראקטיבי ללקוח** - מחשבון פיננסי חי עם גרפים
- **מודל פיננסי מלא** - IRR, NPV, תזרים מזומנים ל-25 שנה
- **תמיכה ביצרנים** - JINKO, FFD, SOLAREDGE

## שימוש

1. פתח `index.html` (לוח בקרה)
2. התחבר עם admin / admin
3. צור הצעת מחיר חדשה
4. העתק את הקישור ושלח ללקוח

## הרצה מקומית

```bash
# כל שרת סטטי יעבוד
npx serve .
# או
python3 -m http.server 8000
```

## אירוח ב-GitHub Pages

האתר מוגש אוטומטית מ-GitHub Pages.
